# 📱 SolveAI Mini App

Mini App для Telegram бота SolveAI.

## 🚀 Быстрый старт

### 1. Откройте в браузере
```
https://YOUR_USERNAME.github.io/solveai-mini-app/
```

### 2. Откройте в Telegram
```
/start → 📊 Статистика → 📈 Открыть статистику
```

## 📁 Файлы

- `index.html` - Основной UI
- `app.js` - Логика и Telegram SDK
- `style.css` - Стили

## 🔧 Настройка

### Для разработчиков

1. Клонируйте репозиторий
2. Внесите изменения
3. Запушьте на GitHub
4. GitHub Pages автоматически обновится

### Переменные окружения (в боте)

```env
WEB_APP_URL=https://YOUR_USERNAME.github.io/solveai-mini-app/
```

## 🎨 Кастомизация

### Изменить цвета
Откройте `style.css` и измените CSS переменные:
```css
:root {
  --tg-theme-button-color: #667eea;
  --tg-theme-button-text-color: #ffffff;
}
```

### Изменить данные
Откройте `app.js` и измените `appData`.
